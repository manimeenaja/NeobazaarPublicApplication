Neobazaar Frontend Application
==============================

This module contains the AngularJS application of Neobazaar.

### Run unit test
 
Please note you must be in the module root using the console.

```
// run unit tests
grunt test:unit

// run midway tests
grunt test:midway

// run e2e tests
grunt test:e2e

// run all tests
grunt test
```
